# Task 1: Email Domain Checker

Скрипт для многоуровневой проверки email-адресов.

## Функционал
1.  **Синтаксический анализ**: Проверка корректности написания адреса через Regex.
2.  **DNS Validation**: Проверка существования домена и наличия MX-записей (Mail Exchanger).
3.  **SMTP Handshake**: Подключение к почтовому серверу и выполнение команды `RCPT TO` для проверки существования конкретного ящика (без отправки письма).

## Запуск
Убедитесь, что зависимости установлены (`pip install dnspython`).

```bash
python main.py <путь_к_файлу>
