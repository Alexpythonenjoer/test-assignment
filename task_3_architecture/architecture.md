# Архитектура системы холодного аутрича (1200+ контактов)

## 1. Общее описание
Для обеспечения высокой доставляемости (Inbox Rate) и защиты репутации доменов используется стратегия **распределенной отправки** с пулом «прогретых» аккаунтов. Основной принцип: «Много доменов — мало писем с каждого».

## 2. Инфраструктура и стек
*   **Backend**: Python (FastAPI/Celery) для управления очередями и логикой.
*   **Database**: PostgreSQL (хранение статусов писем и истории взаимодействий).
*   **Queue**: Redis (брокер задач для асинхронной отправки).
*   **Hosting**: VPS (Hetzner/DigitalOcean) — 2 vCPU, 4GB RAM (~$6-10/мес).

## 3. Почтовый слой и Ротация
Для безопасной работы с базой в 1200+ контактов (с учетом доотправок и цепочек):
*   **Пул доменов**: 15–20 look-alike доменов (например, `get-company.com`, `try-company.co`).
*   **Аккаунты**: 2–3 ящика на домен (итого 45–60 рабочих аккаунтов).
*   **Лимиты**: Не более **30–40 писем в сутки** с одного ящика.
*   **Ротация**: Каждый новый лид назначается на случайный свободный ящик из пула для равномерного распределения нагрузки.

## 4. Прогрев и Технические настройки
*   **DNS**: Обязательная настройка записей **SPF, DKIM, DMARC** для каждого домена.
*   **Warm-up**: Предварительный прогрев доменов (14–21 день) через сервисы (Instantly.ai / Smartlead) для формирования траста у почтовых провайдеров (Google/Outlook).
*   **Spintax**: Использование вариативности текста (шаблоны с заменой синонимов), чтобы письма не выглядели идентичными.

## 5. Мониторинг и Риски

| Риск | Способ закрытия |
| :--- | :--- |
| **Попадание в Blacklist** | Автоматическая проверка через API MxToolbox. Замена домена при попадании в листинг. |
| **Высокий Bounce Rate** | Предварительная валидация базы (скриптом из Task 1) перед загрузкой. |
| **Бан домена провайдером** | Использование разных почтовых провайдеров (Zoho + Google Workspace + MS 365). |

## 6. Оценка стоимости (в месяц)
*   **Домены**: ~$25 (амортизация стоимости регистрации).
*   **Почтовые подписки (Zoho/Google)**: ~$80–120.
*   **Инфраструктура (VPS)**: ~$10.
*   **Итого**: **~$115 – $155 / мес.**
