# Решение тестового задания (Email Checker & Telegram Outreach)

Данный репозиторий содержит выполнение комплексного тестового задания, включающего автоматизацию проверки email-доменов, интеграцию с Telegram API и проектирование архитектуры аутрич-системы.
## 🚀 Запуск скриптов

Каждая задача содержит свой собственный `README.md` с подробными деталями внутри соответствующей директории.

## 📁 Структура проекта
*   `task_1_email_checker/` — Скрипт проверки MX-записей и выполнения SMTP Handshake.
*   `task_2_telegram_bot/` — Интеграция с Telegram для отправки текстов из файлов.
*   `task_3_architecture/` — Техническое описание архитектуры системы на 1200+ адресов.
*   `task_4_blitz/` — Ответы на вопросы по AI-стеку и инструментам разработки.
*   `requirements.txt` — Единый список зависимостей для всех скриптов.
