# Task 2: Telegram Integration

Скрипт для автоматизированной отправки текстового контента из локального файла в Telegram-чат через Bot API.

## 🛠 Предварительная настройка

Для работы скрипта необходимо создать бота и получить идентификатор чата:

1. **Создание бота**:
   - Напишите [@BotFather](https://t.me) в Telegram.
   - Используйте команду `/newbot`, следуйте инструкциям и сохраните **API Token**.
2. **Получение Chat ID**:
   - Напишите созданному боту любое сообщение.
   - Перешлите это сообщение боту [@userinfobot](https://t.me), чтобы узнать свой числовой `ID`.
3. **Настройка окружения**:
   - Создайте в текущей папке файл `.env` (на основе `.env.example`).
   - Впишите туда полученные `TELEGRAM_TOKEN` и `CHAT_ID`.

## 🚀 Запуск

1. Убедитесь, что зависимости установлены:
   ```bash
   pip install requests python-dotenv
   
2. 
   ```
   python bot.py
